    Technický manuál

    Tato aplikace je napojena na projekt ve službě Firebase. Využívá dvě hlavní služby: Firestore Database pro ukládání strukturovaných dat a Cloud Storage pro ukládání souborů (obrázků samolepek).

    Detaily připojení

    • Project ID: central-asset-storage
    • Cloud Storage Bucket: central-asset-storage.appspot.com

    Struktura dat

    Veškerá data aplikace jsou z bezpečnostních důvodů ukládána pod specifickou cestou, která zajišťuje oddělení dat mezi různými aplikacemi.

    • Šablony bannerů jsou uloženy v kolekci Firestore:
      /artifacts/banner-creator-app/public/data/banner_templates
    • Metadata samolepek jsou uložena v kolekci Firestore:
      /artifacts/banner-creator-app/public/data/stickers
    • Soubory samolepek se ukládají do Cloud Storage do složky:
      /artifacts/banner-creator-app/public/stickers/
    ⚡ Quick Deploy 
    # ⚡ Rychlé volby (copy & paste podle situace)

# 1) 1:1 deploy – nic nepřegeneruje, vezme aktuální build
firebase deploy --only hosting:onlineday73

# 2) Nový build + deploy (když se měnil kód / assets)
npm run sync:grid
npm run build
firebase deploy --only hosting:onlineday73

# 2b) Build + deploy i s functions (pokud se měnily)
npm run sync:grid
npm run build
firebase deploy --only hosting:onlineday73,functions

# 3) Preview kanál (např. PR 123, expirace 7 dní)
firebase hosting:channel:deploy pr-123 --expires 7d --only hosting:onlineday73

## 1. Cache problém (nejčastější)
# Hard refresh (vynucený reload):
Cmd+Shift+R (Mac) nebo Ctrl+Shift+R (PC)

# DevTools disable cache:
F12 → Network tab → ☑ "Disable cache" → reload

## 2. Dev server cache
# Restart Vite serveru:
Ctrl+C
npm run dev

## 3. Diagnostika v konzoli
# Zkontroluj element v DOM:
document.querySelector('[data-toplink="super-modul"]')

# Ověř načtený soubor:
location.pathname
fetch(location.href.split('?')[0]).then(r=>r.text()).then(t=>console.log(t.includes('Super Modul')))

## 4. Duplicitní soubory
# Ujisti se, že upravuješ správný soubor:
# ✅ DAY73/grid-app-test.html
# ❌ DAY73/grid-app-test (kopie).html

## 5. URL parametry pro cache-bust:
?v=1234567890&cache=false
